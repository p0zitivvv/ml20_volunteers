ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ ФОТОГРАФИЙ
========================================

В этой папке нужно разместить следующие фотографии:

1. cat.jpg - фотография кошки из приюта (для hero-блока)
2. dog.jpg - фотография собаки из приюта (для hero-блока)
3. help-animal.jpg - фотография волонтёра, помогающего животному (для карточки "Помогаем животным")
4. shelter.jpg - фотография приюта для животных (для карточки "Поддерживаем приюты")
5. kindness.jpg - фотография, показывающая доброту к животным (для карточки "Делаем мир добрее")
6. dog-bubble1.jpg - фотография собаки (для секции "Те, ради кого мы действуем")
7. cat-bubble1.jpg - фотография кошки (для секции "Те, ради кого мы действуем")
8. animal-bubble1.jpg - фотография животного (для секции "Те, ради кого мы действуем")

ВАЖНО:
- Если фотографии отсутствуют, сайт автоматически покажет эмодзи вместо них
- Рекомендуемый размер фотографий: минимум 400x400 пикселей
- Формат: JPG, PNG или WebP
- Фотографии должны быть оптимизированы для веба (не слишком большие файлы)
